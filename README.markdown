#### Usage / Instructions ####

1. Install AHK (https://www.autohotkey.com/)
2. Download (https://github.com/Michal1889/PoEDPSCalc/releases) and extract the zip-file
3. Run **DPS_Calculator.ahk**


|Shortcut|Description|
|---	|---	|
| `ctrl+c` | Check weapon's DPS / Gem Info
